# 再帰反射 専門用語集 (特許情報対応・75ワード版)

再帰反射シートの技術、物理的原理、測定方法、セキュリティ技術、および規格に関連する専門用語を75語網羅した用語集です。主要メーカー（3M, ORAFOL, 日本カーバイド工業等）の特許技術や、国際特許分類（IPC）などの特許情報も含めて解説しています。
