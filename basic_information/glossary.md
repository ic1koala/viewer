# 再帰反射 専門用語集

再帰反射シートの技術、物理的原理、測定方法、および規格に関連する専門用語を網羅した用語集です。

---

## 用語一覧（五十音・アルファベット順）

| 用語名 (日本語/英語) | 読み方 | カテゴリ | 定義・詳細解説 |
| :--- | :--- | :--- | :--- |
| **一時的反射輝度低下**<br>*(Temporary Loss of Retroreflectivity)* | いちじてきはんしゃきどていか | 性能・環境 | 露出レンズ型などで、表面に水滴や泥が付着することでレンズ効果の焦点が狂い、一時的に反射性能が極端に低下する現象。封入型やプリズム型ではこの問題が起きにくい構造になっている。 |
| **入射角**<br>*(Entrance Angle / 記号: $\beta$)* | にゅうしゃかく | 測定幾何学 | 反射シートに当たる光の照射軸と、シート面に対して垂直な基準軸（法線）とがなす角度。カーブの手前や交差点などでは入射角が大きくなるため、大入射角でも明るく反射する「広角性」が求められる。 |
| **右方・左方配向**<br>*(Orientation)* | うほう・さほうはいこう | 光学設計 | プリズム型反射シートにおいて、特定の方向（水平または垂直）からの光に対して反射効率を最適化するため、微細なプリズム素子を傾けて配置すること。 |
| **液中屈折率**<br>*(Refractive Index in Liquid)* | えきちゅうくっせつりつ | 物理・光学 | 液体中に浸された状態におけるビーズやプリズムの屈折率。再帰反射材が雨で濡れた際の光の挙動を計算する際に重要となる。 |
| **観測角**<br>*(Observation Angle / 記号: $\alpha$)* | かんそくかく | 測定幾何学 | 光源から反射シートに届く「照射軸」と、シートから観測者の目に届く「観測軸」とがなす角度。ドライバーの目とヘッドライトの物理的な距離差に対応し、この角度が小さいほど反射光は明るく見える。 |
| **観測軸**<br>*(Observation Axis / 記号: O)* | かんそくじく | 測定幾何学 | 反射シートの基準中心点と、受光器（または観測者の目）の中心とを結ぶ直線。 |
| **基準軸**<br>*(Reference Axis / 記号: R)* | きじゅんじく | 測定幾何学 | 反射シートの表面に対して垂直な線（法線）。入射角の測定基準となる。 |
| **き裂耐性**<br>*(Cracking Resistance)* | きれつたいせい | 物理・耐久性 | 屋外での長期間の使用や極端な温度変化において、反射シートの表面フィルムにヒビや割れ（クラック）が生じないことを示す耐候性能。 |
| **境界角**<br>*(Limit Angle)* | きょうかいかく | 測定幾何学 | 測定系において有効な反射光が得られる最大限界の角度。 |
| **鏡面反射 / 正反射**<br>*(Specular Reflection)* | きょうめんはんしゃ / せいはんしゃ | 物理・光学 | 入射した光が、入射角と等しい反射角の方向（対称な方向）へ一斉に戻る反射。通常の鏡や水面で見られる反射形式。 |
| **極点**<br>*(Pole)* | きょくてん | 物理・光学 | 球形ガラスビーズの頂点および最深部。入射光がビーズ裏面の極点で焦点を結び反射することが再帰反射の理想条件となる。 |
| **比反射輝度係数**<br>*(Coefficient of Retroreflection / 記号: $R_A$)* | ひはんしゃきどけいすう | 測定幾何学 | 反射シートの再帰反射性能を表す標準的な物理量。単位は $cd \cdot lx^{-1} \cdot m^{-2}$。受光器方向への反射光度を、シート面の照度と面積で割って算出する。 |
| **屈折率**<br>*(Refractive Index / 記号: n)* | くっせつりつ | 物理・光学 | 真空（または空気）中の光速と、媒質中の光速の比。ガラスビーズ方式では、空気中で再帰反射させるための理想的なビーズ屈折率は $n \approx 2.0$ である。 |
| **クリスタルグレード**<br>*(Crystal Grade)* | くりすたるぐれーど | 製品・メーカー | 日本カーバイド工業（NCI）が提供する最上位クラスのカプセルプリズム型再帰反射シートの製品シリーズ名。 |
| **広角性**<br>*(Angularity)* | こうかくせい | 性能・環境 | 光が反射シートに対して斜めから入射した際（大入射角）、あるいは観測者が光源から離れた位置にいる際（大観測角）でも、高い反射輝度を維持できる特性。 |
| **降雨時視認性**<br>*(Wet Performance)* | こううじしにんせい | 性能・環境 | 雨天時にシート表面が水膜で覆われた状態でも、反射輝度を損なわずに光源へ光を戻すことができる性能。 |
| **コーナーキューブ**<br>*(Corner Cube)* | こーなーきゅーぶ | 光学設計 | 立方体の角を切り取った三面直交の錐体構造。3つの直交する平面で光が全反射を繰り返すことで、入射した光が正確に逆方向へと戻る特性を持つ。プリズム型反射シートの基本素子。 |
| **散乱反射 / 乱反射 / 拡散反射**<br>*(Diffuse Reflection)* | さんらんはんしゃ / らんはんしゃ | 物理・光学 | 入射した光が、反射体表面の凹凸などによって様々な方向へ不規則に散乱する反射。通常の白い壁や衣類など、多くの物体で見られる。 |
| **三面体プリズム**<br>*(Trihedral Prism)* | さんめんたいぷりずむ | 光学設計 | コーナーキューブと同義。3つの直交する平面反射面を持つ三角錐型の素子。 |
| **照射軸**<br>*(Illumination Axis / 記号: I)* | しょうしゃじく | 測定幾何学 | 光源（投光器）の中心点と、反射シートの基準中心点とを結ぶ直線。 |
| **スネルの法則**<br>*(Snell's Law)* | すねるのほうそく | 物理・光学 | 光が異なる屈折率を持つ2つの媒質の境界を通過する際、入射角 $\theta_1$ と屈折角 $\theta_2$、およびそれぞれの屈折率 $n_1, n_2$ の間に成り立つ関係式（$n_1 \sin \theta_1 = n_2 \sin \theta_2$）。ビーズ型反射材の焦点計算の基礎となる。 |
| **全反射**<br>*(Total Internal Reflection)* | ぜんはんしゃ | 物理・光学 | 光が屈折率の大きい媒質から小さい媒質へと進む際、入射角が「臨界角」より大きくなると、境界で光が一切透過せず、すべて反射される現象。プリズム型反射シートはこの全反射を利用して極めて高い輝度を実現する。 |
| **ダイヤモンドグレード**<br>*(Diamond Grade)* | だいやもんどぐれーど | 製品・メーカー | 3M社が提供する、最高輝度かつ広角性に優れたフルキューブプリズム型反射シートの登録商標・製品シリーズ名。 |
| **単層構造 / ソリッド構造**<br>*(Solid / Single Layer Structure)* | たんそうこうぞう / そりっどこうぞう | 光学設計 | プリズムの裏面を直接保護樹脂や接着剤で充填し、空気層を持たない構造。オラフォルの製品に多く見られ、切断後に雨水の侵入を防ぐための端面シール加工（目封じ）が不要という強みを持つ。 |
| **端面シール / 目封じ**<br>*(Edge Sealing)* | たんめんしーる / めふうじ | 施工・加工 | カプセル構造（空気層を持つ）の反射シートを切断した際、切断面から水分や汚れが空気層へ侵入するのを防ぐために、熱や溶剤で端部を融着・密閉する加工処理。 |
| **破壊性反射シート**<br>*(Destructible Reflective Sheet)* | はかいせいはんしゃしーつ | セキュリティ | 貼り付けた対象から剥がそうとすると、基材層が細かく裂けて自己破壊される構造を持つセキュリティ用反射シート。ナンバープレートの偽造防止や公式ステッカーの転用防止に用いられる。日本カーバイド工業（NCI）の特許技術。 |
| **バッテンバーグ・マーキング**<br>*(Battenburg Marking)* | ばってんばーぐ・まーきんぐ | 性能・環境 | 欧州などの警察・救急車両によく見られる、視認性の高いチェッカー模様（黄色と青/赤など）の反射デザイン。遠方からの即時認知性を高める。 |
| **フルキューブプリズム**<br>*(Full Cube Prism)* | ふるきゅーぶぷりずむ | 光学設計 | 従来の三面体プリズム（コーナーキューブ）で生じていた「幾何学的に全反射に寄与しない余剰領域（非アクティブエリア）」を排除し、有効反射面積をほぼ100%にした3M独自の超高精度プリズム素子。 |
| **臨界角**<br>*(Critical Angle)* | りんかいかく | 物理・光学 | 全反射が発生し始める最小の入射角。媒質1（屈折率 $n_1$）から媒質2（屈折率 $n_2$、ただし $n_1 > n_2$）へ光が進むとき、$\theta_c = \arcsin(n_2 / n_1)$ で表される。 |
| **ECE 104** | いーしーいーいちまるよん | 規格・規制 | ヨーロッパにおける大型商用車両（トラックやトレーラーなど）の側面・後部に貼り付ける、視認性向上のための反射マーキングテープの安全基準および技術規格。 |
| **JIS Z 8714** | じすぜっとはちなないちよん | 規格・規制 | 日本産業規格（JIS）において、再帰反射安全表示の物理測定方法、比反射輝度係数の算出方法、安全色彩および反射性能のクラス基準を定めた規格。 |
