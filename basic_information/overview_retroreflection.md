# 再帰反射の物理的原理と測定幾何学

再帰反射（Retroreflection）は、入射した光が反射体の表面で屈折・反射を繰り返し、光がやってきた光源の方向へと正確に戻る光学現象です。本ドキュメントでは、再帰反射の2大方式（ガラスビーズ型・プリズム型）の物理的・数学的メカズムと、その性能を規定する測定幾何学について詳細に解説します。

---

## 1. ガラスビーズ型（Glass Bead Type）の反射メカニズム

ガラスビーズ型は、微小な球状のガラス粒子をレンズとして利用し、その背面に配置された反射層（金属蒸着膜など）と組み合わせて再帰反射を実現する方式です。

### 屈折と集光の幾何光学

球形のガラスビーズに平行光線が入射した際、光がビーズの裏面（極点）に焦点を結ぶ条件を考えます。
光線がビーズ表面で屈折する様子は、スネルの法則（Snell's Law）によって決定されます。

\[ n_1 \sin \theta_1 = n_2 \sin \theta_2 \]

ここで、外部媒質（空気）の屈折率を $n_1 = 1.0$、ガラスビーズの屈折率を $n_2 = n$ とします。

球面の屈折公式（近軸近似：光軸に極めて近い光線）において、物点距離を $s$、像点距離を $s'$、球面の曲率半径を $R$ とすると、以下の関係式が成り立ちます。

\[ \frac{n_1}{s} + \frac{n}{s'} = \frac{n - n_1}{R} \]

平行光線が入射する場合、物点距離は無限遠（$s = \infty$）となるため、左辺第1項は $0$ となり、式は以下のように簡略化されます。

\[ \frac{n}{s'} = \frac{n - 1}{R} \implies s' = \frac{n}{n - 1}R \]

ビーズの頂点（入射点）から裏面（反射面）までの距離は、直径に等しいため $2R$ です。像点（焦点）がちょうどビーズの裏面に位置する条件（$s' = 2R$）を求めると：

\[ 2R = \frac{n}{n - 1}R \implies 2(n - 1) = n \implies n = 2.0 \]

したがって、**空気中においては屈折率がちょうど $2.0$ のガラスビーズを使用すると、入射した平行光線はビーズの裏面に焦点を結び、そこに施された反射層で反射し、再びビーズを通って元の光源方向へ完全に平行に戻る**ことになります（これを「露出レンズ型」の基本設計と呼びます）。

#### 実用上の補正（封入レンズ型）
実際の製品（封入レンズ型など）では、ガラスビーズの前面および背面が透明な樹脂層（屈折率 $n \approx 1.5$ 程度）で覆われています。この場合、外部媒質が空気ではなく樹脂となるため、最適なビーズの屈折率は約 **1.9** 程度に設計されます。
また、露出レンズ型であっても、近軸外の光線（球面収差）や製造上の許容誤差を考慮し、最適値は **2.2** 前後に調整されるのが一般的です。

## 2. プリズム型（Prism / Corner Cube Type）の反射メカニズム

プリズム型は、立方体の角を切り取った形状である「コーナーキューブ（Corner Cube）」素子を敷き詰めた構造です。鏡の全反射を利用するため、ガラスビーズ型に比べて反射効率が極めて高いのが特徴です。

### 直交三面鏡によるベクトル反射計算

コーナーキューブは、互いに直交する3つの平面鏡（面A: $x=0$, 面B: $y=0$, 面C: $z=0$）で構成される光学系としてモデル化できます。

入射光線の方向ベクトルを $\vec{u} = (u_x, u_y, u_z)$ とします。
一般に、法線ベクトル $\vec{n}$ を持つ平面鏡で光線が反射した後の方向ベクトル $\vec{u}'$ は、鏡面反射の法則より以下のように表されます。

\[ \vec{u}' = \vec{u} - 2(\vec{u} \cdot \vec{n})\vec{n} \]

光線が3つの直交する平面でそれぞれ1回ずつ、計3回反射するプロセスを順に計算します。

1. **第1反射面（面A: $x=0$, 法線 $\vec{n}_x = (1, 0, 0)$）**
   \[ \vec{u}' = (u_x, u_y, u_z) - 2 \left( (u_x, u_y, u_z) \cdot (1, 0, 0) \right) (1, 0, 0) \]
   \[ \vec{u}' = (u_x, u_y, u_z) - 2u_x(1, 0, 0) = (-u_x, u_y, u_z) \]
   （$x$ 成分のみ符号が反転する）

2. **第2反射面（面B: $y=0$, 法線 $\vec{n}_y = (0, 1, 0)$）**
   \[ \vec{u}'' = \vec{u}' - 2(\vec{u}' \cdot \vec{n}_y)\vec{n}_y \]
   \[ \vec{u}'' = (-u_x, u_y, u_z) - 2u_y(0, 1, 0) = (-u_x, -u_y, u_z) \]
   （$y$ 成分も符号が反転する）

3. **第3反射面（面C: $z=0$, 法線 $\vec{n}_z = (0, 0, 1)$）**
   \[ \vec{u}''' = \vec{u}'' - 2(\vec{u}'' \cdot \vec{n}_z)\vec{n}_z \]
   \[ \vec{u}''' = (-u_x, -u_y, u_z) - 2u_z(0, 0, 1) = (-u_x, -u_y, -u_z) \]
   （すべての成分の符号が反転する）

最終的に得られる反射光線の方向ベクトルは：

\[ \vec{u}''' = -\vec{u} \]

これは、**入射した光が、その入射角度（入射ベクトル $\vec{u}$）に依存せず、常に完全に逆方向（$-\vec{u}$）へ向かって反射して戻る**ことを数理的に示しています。

#### 臨界角と全反射条件
プリズム型において、光はガラスやアクリルなどの透明樹脂内部から空気との境界（プリズムの斜面）へ入射し、**全反射（Total Internal Reflection）**します。
全反射が起こる臨界角 $\theta_c$ は、樹脂の屈折率を $n_p$ とすると、スネルの法則より次のように定まります：

\[ \theta_c = \arcsin\left(\frac{1.0}{n_p}\right) \]

アクリル樹脂（$n_p \approx 1.49$）の場合、臨界角 $\theta_c \approx 42.2^\circ$ です。コーナーキューブへの入射角がこの臨界角を超えると、光は裏面へ透過してしまい（光漏れ）、反射輝度が低下します。このため、プリズム素子の幾何学的配置やシート内の傾き（配向）を工夫し、広い入射角に対しても全反射条件を維持する設計がなされています。

## 3. 再帰反射の測定幾何学（JIS Z 8714 / CIE規格）

再帰反射材料の性能を客観的に評価するため、国際照明委員会（CIE）および日本産業規格（JIS Z 8714）では、幾何学的な位置関係（角度）が厳密に定義されています。

### 主要な幾何パラメータ

夜間の路上において、自動車のヘッドライト（光源）、ドライバーの目（観測者）、道路標識（反射シート）の位置関係をシミュレートするため、以下の要素が用いられます。

1. **照射軸（Illumination Axis / 記号: I）**
   * 光源の中心と、反射シートの基準中心を結ぶ直線。
2. **観測軸（Observation Axis / 記号: O）**
   * 反射シートの基準中心と、受光器（観測者の目）の中心を結ぶ直線。
3. **基準軸（Reference Axis / 記号: R）**
   * 反射シートの表面に対して垂直な線（法線）。
4. **観測角（Observation Angle / 記号: $\alpha$）**
   * 照射軸と観測軸がなす角度。
   * **物理的意味:** ドライバーの目とヘッドライトの「物理的な高低差・距離」に対応します。距離が離れるほど、または大型車になるほど観測角は大きくなります。通常、測定は $12'$ ($0.2^\circ$)、$20'$ ($0.33^\circ$)、$2^\circ$ などで行われます。観測角が小さいほど、反射光は集中して明るく見えます。
5. **入射角（Entrance Angle / 記号: $\beta$）**
   * 照射軸と基準軸がなす角度。
   * **物理的意味:** 反射シートに対して光がどれだけ「斜めから」当たっているかを表します。カーブの入り口にある標識や、道路脇のポールなどは入射角が大きくなります。通常、測定は $5^\circ$、$30^\circ$、$40^\circ$ などの条件で行われます。

### 比反射輝度係数（Coefficient of Retroreflection: $R_A$）

再帰反射性能を表す最も代表的な物理量は、**比反射輝度係数（または再帰反射係数）$R_A$** です。

\[ R_A = \frac{I}{E_\perp \cdot A} \]

*   **$I$**: 観測者（受光器）方向への反射光度（単位: カンデラ $cd$）
*   **$E_\perp$**: 反射シート面に垂直な面における照度（単位: ルクス $lx$）
*   **$A$**: 反射シートの有効面積（単位: 平方メートル $m^2$）
*   **単位:** $cd \cdot lx^{-1} \cdot m^{-2}$ (または $cd/lx/m^2$)

この値が大きいほど、少ない光でより明るく光源に光を戻せる、優れた再帰反射シートであることを示します。プリズム型はビーズ型と比較して、この $R_A$ の値が数倍〜十数倍高くなります。
